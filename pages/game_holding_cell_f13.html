Holding Cell F13

Holding Cell F13 is a 1st-person real-time point-and-click text adventure game. You are placed in the role of a spaceship pilot who wakes up on an unknown space station with no idea how you got there. A disaster of some kind happens and you must escape the station before it's too late.

The game is played using keyboard and mouse, you can type commands which will cause your character to perform an action of some kind. But you can also click words using your mouse, left mouse button adds the word to a command and right button opens a menu with ways of interacting with the object.

The screen is split up into several parts; on the right items you are currently carrying are shown, on the left there are two boxes, the top shows the room you're in or the object you're examining. Below that is a box that shows actions you have taken and events that are happening in real time.

Genre:Text Adventure
Players:1
Controls:Keyboard; Mouse
Made With:XNA; C#
